# 中區資策會 - Git 與 GitHub

6小時 = 2節

1. Git
  - Git觀念與總覽
  - 安裝與設定 Git 組態
  - 建立 Repository (數據庫)
  - 建立與切換 branch (分支)
  - 觀察/比較/合併 branch (分支)
  - 版本衝突模擬與因應對策

2. GitHub
  - 在 GitHub 建立一個新的 repository (數據庫)
  - 複製遠端 GitHub 數據庫到工作環境(近端)
  - 提取 GitHub 的內容到工作環境(近端)
  - 新增/修改檔案，上傳回 GitHub
  - 解決遠端與近端版本衝突問題

3. 實例研討專案版本管理計畫
